# Architecture Decision Records

Os ADRs registram decisões estruturais da plataforma, seu contexto, alternativas e consequências. Eles devem ser revisados quando premissas de negócio, risco, custo ou tecnologia mudarem.

| ADR | Decisão | Status |
|---|---|---|
| [ADR-001](001-mcp-vs-rest.md) | MCP para tool calling governado; REST para APIs de domínio | Aceito |
| [ADR-002](002-persistent-memory.md) | Memória persistente apenas quando houver valor e base legal | Aceito |
| [ADR-003](003-agent-gateway.md) | Agent Gateway como ponto de entrada e enforcement | Aceito |

## Modelo

Cada ADR contém contexto, decisão, alternativas, consequências, riscos e critérios de revisão.
