# MCP Registry

## Responsabilidades

- Catálogo de MCP Servers
- Descoberta de ferramentas
- Versionamento
- Políticas de acesso
- Governança de tools

## Metadados

- Tool Name
- MCP Server
- Owner
- Version
- Required Scopes

## APIs

GET /tools
GET /tools/{toolId}
POST /tools

## Integrações

- Agent Runtime
- Governance Service
- Corporate MCP Servers
