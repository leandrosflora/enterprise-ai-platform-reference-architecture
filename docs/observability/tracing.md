# Tracing

## Objetivo

Garantir rastreabilidade ponta a ponta das execuções dos agentes.

## Padrão

OpenTelemetry

## Trace Principal

Agent Invocation

### Spans

- Agent Execution
- Memory Retrieval
- Knowledge Retrieval
- Model Call
- Tool Call
- Evaluation

## Identificadores

- traceId
- spanId
- correlationId
- causationId

## Dashboards

- Latência
- Taxa de erro
- Uso de ferramentas
- Uso de modelos
- Custos por agente
