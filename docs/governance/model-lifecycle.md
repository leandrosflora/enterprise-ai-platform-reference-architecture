# Model Lifecycle

## Ciclo de Vida

```text
Candidate
  ↓
Evaluation
  ↓
Approved
  ↓
Production
  ↓
Monitoring
  ↓
Retired
```

## Critérios

### Avaliação

- Qualidade
- Custo
- Latência
- Segurança

### Produção

- Observabilidade ativa
- Auditoria habilitada
- FinOps habilitado

### Retirada

- Modelo obsoleto
- Alto custo
- Baixa qualidade
- Problemas regulatórios
